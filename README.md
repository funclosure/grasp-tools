# Grasp Tools

*Mind extension without outsourcing*

## What is this?

Grasp Tools is a design research repo for exploring tools that extend human cognition without hollowing out the human who is doing the thinking.

The core question is simple:

> How do we build cognitive tools that increase grasp, rather than merely increasing access to answers?

This project sits at the intersection of:
- humane interfaces
- extended mind
- embodied cognition
- relevance realization
- participatory sensemaking
- AI and cognitive scaffolding

## Core thesis

Some tools extend mind by recruiting more of the human being into the act of knowing.
Other tools outsource cognition by delivering useful outputs while bypassing the growth of understanding, salience, skill, and participation.

Grasp Tools exists to study, design, and eventually build the first kind.

## Guiding distinction

### Participatory extension
Tools that leave the human more capable, more oriented, and more able to think.

### Substitutive outsourcing
Tools that maximize convenience but leave the human thinner, more passive, or less able to proceed without the system.
